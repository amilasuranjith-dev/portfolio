# Modern Developer Portfolio

A high-performance, visually stunning developer portfolio built with **Next.js 14**, **TypeScript**, and **Framer Motion**. This project features smooth scroll animations, interactive 3D/particle effects, and a fully responsive design tailored for a premium user experience.

![Portfolio Preview](/public/portfolio_img.png)

## ✨ Key Features

- **🚀 Next.js 14 (App Router)**: Optimized for performance and SEO.
- **🎨 Tailwind CSS + Framer Motion**: Advanced animations including scroll reveals, magnetic buttons, and tilt effects.
- **🌌 Interactive Background**: Custom canvas-based particle galaxy that reacts to mouse movement.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **🌗 Dark Mode**: Deep space aesthetic with carefully balanced contrast for readability.
- **🏗️ Component Architecture**: Highly modular structure with reusable UI components and custom hooks.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📂 Project Structure

```text
src/
├── app/            # Next.js App Router (Layouts, Pages, Globals)
├── components/     # UI, Layout, and Section-specific components
├── data/           # Portfolio content (Experience, Projects, Skills)
├── hooks/          # Custom React hooks (Mouse position, Scroll progress)
├── utils/          # Animation configurations and helper functions
└── public/         # Static assets and images
```

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm / yarn / pnpm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amilasuranjith-dev/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
